# MathContentUE
## Project Description
The essence of the project is to identify and use mathematical techniques on the example of using Unreal Engine 5  
List of examples:  
- Radial Trigger  
- Look At Trigger  
- Space Transform  
- Laser Reflection

Description: Calculation of the distance by the radius of the circle
